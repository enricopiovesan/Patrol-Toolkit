# OSM Extractor Roadmap 1.5.0

## Objective
Provide machine-readable pack summaries for automated review pipelines and CI reporting.

## Capabilities
- `summarize-pack --json` output mode.
- Structured summary payload includes:
  - resort id/name,
  - schema version,
  - run/lift/tower counts.
- Existing human-readable `summarize-pack` output remains unchanged.

## Acceptance Criteria
1. `summarize-pack --json` returns valid JSON with deterministic fields.
2. Summary counts match the validated pack content.
3. Existing text summary behavior remains backward compatible.
4. Tests cover structured summary generation.
