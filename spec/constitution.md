# Patrol Toolkit Constitution

## Purpose
Patrol Toolkit exists to support mountain patrol operations by providing reliable, offline-first spatial intelligence tools.

## Primary Principle
Radio remains the primary communication channel at all times.

This application must never:
- Replace radio communication.
- Become safety critical infrastructure.
- Require network connectivity to function.

## Design Principles
- Offline First
- Deterministic Logic
- Data Driven
- Simplicity under pressure
- Assistive only, never authoritative
- One active resort context per session
