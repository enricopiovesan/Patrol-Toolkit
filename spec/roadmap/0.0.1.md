# Roadmap Version 0.0.1

## Objective
Convert GPS location into concise patrol-grade radio phrase fully offline.

## Personas
- Patroller under radio pressure

## Capabilities
- Offline PWA
- Resort Pack loader
- GPS dot
- Deterministic radio phrase generator
- One-tap copy short phrase

## Out of Scope
- Sweep workflow
- Shared state
- Dispatch integration

## Acceptance Criteria
1. Works fully offline.
2. Generates phrase within 2 seconds.
3. Includes run name when inside geometry.
4. Includes nearest lift tower when within threshold.
5. Copy button functional.

## Exit Criteria
- Tested during real patrol day.
- Used successfully in at least one live radio call.
- Feedback collected from patrol members.
